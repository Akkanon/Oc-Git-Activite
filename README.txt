-- Activité Git & GitHub --
-Activité consistant à créer en local ou en ligne un repository comprenant deux fichiers avec plusieurs commit chacun

-Créer le rertoire
-L'intitaliser et le placer sur gitHub
-Créer ou importer deux fichiers
-Les modifier par le biais d'un commit au moins quatre fois
-Placer l'adresse du repository dans le document texte prévu à cet effet et l'envoyer en zippé
